== Installation ==

= Using The WordPress Dashboard =

1. Navigate to the 'Add New' in the plugins dashboard
2. Search for 'Eazy Plugin Manager'
3. Click 'Install Now'
4. Activate the plugin on the WordPress plugin dashboard


= Uploading in WordPress Dashboard =

1. Navigate to the 'Add New' in the plugins dashboard
2. Navigate to the 'Upload' area
3. Select `plugins-on-steroids.zip` from your computer
4. Click 'Install Now'
5. Activate the plugin in the WordPress plugin dashboard

= Using FTP =

1. Download `plugins-on-steroids.zip`
2. Extract the `plugins-on-steroids` directory to your computer
3. Upload the `plugins-on-steroids` directory to the `/wp-content/plugins/` directory
4. Activate the plugin in the WordPress plugin dashboard
